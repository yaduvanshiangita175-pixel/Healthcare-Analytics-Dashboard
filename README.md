# 🏥 Healthcare Analytics Dashboard

## 🔍 Objective
To analyze hospital performance, patient data, and operational metrics using data visualization techniques.

---

## 🛠️ Tools Used
- Power BI  
- Tableau  
- Excel  
- SQL  

---

## 📈 Key Metrics
- Patient Visits  
- Treatment Cost  
- Follow-up Rate  
- Abnormal Lab %  

---

## ⚙️ Process
- Data Cleaning  
- Data Transformation  
- Dashboard Development  
- Insight Generation  

---

## 💡 Key Insights
- Identified top diagnosed conditions  
- Analyzed hospital performance trends  
- Evaluated doctor workload and patient flow
